# Wordle Game

## Prérequis
- Node.js (16+)
- npm

## Installation
1. Cloner le dépôt
2. Installer les dépendances : `npm install`

## Commandes
- Démarrer le développement : `npm run dev`
- Lancer les tests : `npm test`
- Générer la couverture de code : `npm run test:coverage`
